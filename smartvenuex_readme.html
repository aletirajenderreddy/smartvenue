Getting Started

Up and running in 4 steps

Clone, configure, and launch — the full stack runs locally in under 5 minutes.

1
Clone the repository
Fork or clone SmartVenueX to your local machine.
bash 
git clone https://github.com/YOUR_USERNAME/smartvenuex.git
cd smartvenuex
2
Install dependencies
Install all frontend packages via npm.
bash 
npm install
3
Configure environment variables
Create a .env.local file in the root directory with your credentials:
.env.local 
# Supabase Configuration
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

# Google Gemini AI
VITE_GEMINI_API_KEY=your_gemini_api_key
VariableDescriptionSource
VITE_SUPABASE_URLYour Supabase project URLRequired
VITE_SUPABASE_ANON_KEYSupabase anonymous public keyRequired
VITE_GEMINI_API_KEYGoogle AI Studio API keyRequired
4
Launch the dev server
Start the Vite development server — it opens automatically at localhost:5173.
bash 
npm run dev
